
安装：

```bash
npm install -g @fast-learn/core
# OR
yarn global add @fast-learn/core
```

创建项目

```bash
fast-learn init 
```

发布项目

```bash
fast-learn publish
```

## More

清空缓存：

```bash
fast-learn clean
```

DEBUG 模式：

```bash
fast-learn --debug
```

指定本地包：

```bash
fast-learn init --packagePath /Users/sam/Desktop/fast-learn/packages/init/
```
